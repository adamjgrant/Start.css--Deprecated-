Boxes

Boxes are the most versatile component of the Start library. These can function as navigation menus, toolboxes, or any generic container.

Navigation Boxes

Boxes for horizontal navigation should be filled with unordered lists (``ul``)

  • Item
  • Item
<nav class="s-box">
  <ul>
    <li>
      <a href="#">Item</a>
    </li>
    <li>
      <a class="s-btn" href="#">Item</a>
    </li>
  </ul>
</nav>

Right-aligned

Use .s-right on the child ul to align right. li.s-box-heading will give you a formatted title.

Growls

Growls are unobtrusive pop up messages called via JavaScript. A basic call might look like this:

$.sGrowl({
   icon: 'save',
   title: 'Saved',
   text: 'Your document has been saved'  
 })

Parameters

Parameter Default Description
icon — Icon to appear before the heading. Choose from the Font Awesome 4 library without the preceding "fa-"
title — Bold-faced title to appear on the first line.
text — Main text under the title. HTML allowed.
delay (Infinite) Delay in miliseconds before dismissing growl. Ignore this parameter to keep growl permanent.
container #s-growls HTML element where growl should be sent. Write in CSS notation.
color green Color of growl (red/orange/yellow/green/blue/violet).
class — Additional classes to apply to growl. Separate multiple classes with a space.

Advanced

Here is a growl that will appear in red in the container below and disappear after 3 seconds.

Growl will appear above.

$.sGrowl({
  color: 'red', 
  icon: 'book',
  title: 'Test',
  text: 'Testing...1, 2, 3', 
  delay: 3000, 
  container: '#example-container'
})
